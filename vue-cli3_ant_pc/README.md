# vue-cli3_ant_pc
> 使用yarn

## 安装
```
yarn install
```

### 开发运行
```
yarn run serve
```

### 打包构建
```
yarn run build
```

### 运行测试
```
yarn run test
```

### 运行代码检测
```
yarn run lint
```

### 运行我的单元测试
```
yarn run test:unit
```
